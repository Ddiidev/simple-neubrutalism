Getting Started

A lightweight, zero-dependency CSS framework for bold neubrutalism interfaces.

Installation

HTML
<link rel="stylesheet" href="neubrutalism.css">

<!-- For modal, tabs, and custom datetime -->
<script src="tags/modal.js"></script>
<script src="tags/tabs.js"></script>
<script src="tags/datetime.js"></script>

Modal

Full-screen overlay with grid-pattern background and JS API.

Preview

Modal Title

This is a modal with neubrutalism grid-pattern background.

HTML
<button class="openModal-demo"
  target-modal="my-modal">Open</button>

<div class="modal" id="my-modal">
  <div class="modal-content">
    <button class="close-modal"
      xclose-modal>&times;</button>
    <h3>Title</h3>
    <p>Content</p>
  </div>
</div>
JS
// Include modal.js:
// <script src="tags/modal.js"></script>

// Manual control:
const modal = document.getElementById('my-modal');
modal.setAttribute('open', '');    // open
modal.removeAttribute('open');     // close

// Callbacks via attributes:
// action-open-callback="onOpen()"
// action-close-callback="onClose()"
